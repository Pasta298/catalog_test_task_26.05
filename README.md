# Сatalog Test Task

Frontend application built with Vite + React.

## Getting started

### 1. Clone the repository

```bash
git clone https://github.com/Pasta298/catalog_test_task_26.05
```

### 2. Navigate to project folder

cd catalog_test_task

### 3. Install dependencies

Using npm:

npm i or npm install

### 4. Start development server

npm run dev

### 5. Open it in browser via

http://localhost:5173 or by ctrl + clicking on address in terminal
